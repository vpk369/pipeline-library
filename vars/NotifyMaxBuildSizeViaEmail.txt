Usage:

```
@Library('utils') _

pipeline {
    agent {
        label 'master'
    }
 
    stages {
        stage('stage0') {
            steps {
                NotifyMaxBuildSizeViaEmail(MAX_BUILD_SIZE_IN_MB:'133')
            }
        }
    }
    

 
}

```
